# ComicReader
Tired of slow readers ruining your experience? Welcome to ComicReader, the site optimized for .cbz and .cbr files. Designed by and for comic and manga lovers, our minimalist interface lets you focus on the only thing that matters: enjoying a good story.

Link: https://ovejoide.github.io/ComicReader/
